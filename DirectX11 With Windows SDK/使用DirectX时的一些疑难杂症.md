# 使用DirectX时的一些疑难杂症

## X5589 Invalid const register num: 155. Max allowed is 31

这是由于使用的常量寄存器超出了限制，不同的着色器模型支持的数量不同，这时提高着色器模型即可。

寄存器类型、数字和子组件声明。

| 类型 | 注册说明         |
| :--- | :--------------- |
| b    | 常量缓冲区       |
| t    | 纹理和纹理缓冲区 |
| c    | 缓冲区偏移量     |
| S    | 取样器           |
| u    | 无序访问视图     |
