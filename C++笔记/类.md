# 类

一种用户定义的引用[数据类型

## 类的基本概念

> 定义一个类：
>
> ```c++
> class Player
> {
> public:	//访问权限
> 
> 	int x, y;
> 	int speed;	//属性
> 
> 	void Move(int ax, int ay)	//行为
> 	{
> 		x += ax * speed;
> 		y += ay * speed;
> 	}
> };
> ```
>
> 使用类实例化一个对象：
>
> ```c++
> Player player0;
> ```

> ## 访问权限
>
> **一个类的所有成员默认情况下是私有成员**
>
> > + private（私有成员）
> > 	只有类中的函数才能访问类中的变量。
>
> > + protected(保护成员)
>
> > + public(公有成员)
> > 	类之外的任何地方都可访问类中的变量

## 类和结构体的对比

> 1. 可见度的区别：
> 	类中成员默认为私有成员，结构体中成员默认为公有成员。
