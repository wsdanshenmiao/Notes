# 链表

## 特征

> + **n个节点离散分配。**
> + **彼此通过指针相连。**
> + **每一个节点只有一个前驱节点，一个后续节点。**
> + **首节点没有前驱节点，尾节点没有后续节点。**
>
> **专业术语：**
>
> + 首节点：第一个有效节点。
> + 尾节点：最后一个有效节点。
> + **头节点：第一个有效节点之前的节点，并不存放有效数据，之时为了方便对指针操作。**
> + **头指针：指向头节点的指针变量，==对链表进行处理只需要知道头指针==**。
> + 尾指针：指向尾节点的指针变量。

## 链表的分类

> 1. 单链表：每个节点有一个指针域。
> 2. 双链表：每个节点有两个指针域，分别指向前一个节点和后一个节点。
> 3. 循环链表：能通过一个节点找到所有节点。
